Custom Directive :

1.Creating a custom directive in Angular is a great way to encapsulate reusable functionality that can be applied to DOM elements. 
Below is a simple practical example that demonstrates how to create a custom directive that changes the background color of an element when the mouse hovers over it.

Step-by-Step Guide for Creating a Custom Directive
1. Create a New Directive
You can use Angular CLI to generate a directive:

ng generate directive highlight
This will create two files:

highlight.directive.ts (directive logic)
highlight.directive.spec.ts (unit test file)
